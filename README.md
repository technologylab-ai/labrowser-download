# LaBrowser Downloads

Public release binaries for participants in [LaBrowser](https://labrowser.app) studies.

Use the download links from your study join page when possible, or visit the [Releases](https://github.com/technologylab-ai/labrowser-download/releases) page directly.

This repo hosts installers only. Product docs, example studies, and onboarding guidance live at [labrowser.app](https://labrowser.app).
